Detailed Description

Weighted angular symmetry function (type 13)

Parameter string:

<element-central> 13 <eta> <rshift> <lambda> <zeta> <rcutoff>

where

  • <element-central> ..... element symbol of central atom
  • <eta> .................
  • <rshift> ..............
  • <lambda> ..............
  • <zeta> ................
  • <rcutoff> .............
